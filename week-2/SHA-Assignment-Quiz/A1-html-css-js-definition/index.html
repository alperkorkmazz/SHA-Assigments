What is Html?

A brief definition

Html is the abbreviation of the Hypertext Markup Language. In technical meaning, it can also defined as main structure of a web page. This Markup language is known by browsers and converted to visual(not too completely but mainly). In many cases html pages are statical unless supported by css and javascript.

What is CSS?

Visual and structural support to html

From colorization to font properties, from paragraph styles to main element locations can be organized by css files. Briefly, one can say that css files are responsible from styling of the html elements used in the related page.
